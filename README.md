# AI Screenshot

A frontend web application for taking, managing, or working with screenshots using an AI-powered workflow.

🔗 **Live Demo:** https://ai-screenshot-eight.vercel.app/
frontend
 

---

## Overview

This repository contains a TypeScript + React frontend built with Vite.  
It uses a modern UI/tooling stack and is configured with Tailwind CSS and ESLint.

Current project structure shows the app living inside:

- `frontend/`

## Getting Started

### Prerequisites

Make sure you have installed:

- **Node.js** (v18+ recommended)
- **npm** (comes with Node.js)

### Installation

```bash
git clone https://github.com/Udaya30/Ai-Screenshot.git
cd Ai-Screenshot/frontend
npm install
```

### Run Development Server

```bash
npm run dev
```

Then open the local URL shown in your terminal (typically `http://localhost:5173`).

---

## Build for Production

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

---

## Linting

```bash
npm run lint
```

## Deployment

The app is deployed on **Vercel**:  
https://ai-screenshot-eight.vercel.app/

For your own deployment:

1. Push your fork/repo to GitHub.
2. Import the project into Vercel.
3. Set the root directory to `frontend` (if needed).
4. Deploy.
